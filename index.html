What is a Frequency Table?

A frequency table summarises how often each unique value appears in a dataset. It displays counts (n), relative frequencies (%), and cumulative percentages to describe the distribution of categorical or discrete variables.

When to Use

  • Summarising categorical survey responses
  • Describing disease case distributions
  • Tabulating diagnostic test outcomes
  • Initial data exploration and quality checks

Output Includes

  • Frequency (n): Count of each category
  • Relative Frequency (%): Proportion of total
  • Cumulative %: Running total percentage
  • Bar chart: Visual distribution
